# Example Client

## Dart/Flutter

Dart is bundled in the Flutter SDK, which is available as a snap (awesome!).

To install:

```
sudo snap install flutter --classic
flutter sdk-path
flutter upgrade
flutter doctor
flutter doctor --android-licenses
flutter config --no-analytics
which flutter dart
```

To run:

```
flutter pub get
dart test_client.dart
```
